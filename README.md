# AI Learning Materials

This repository contains comprehensive study materials for various AI and machine learning topics, organized by subject areas and skill levels.

## Directory Structure

### 01 - Foundations & Core Concepts
- [0101-Statistical-Thinking-and-Applications-Fundamentals](./0101-Statistical-Thinking-and-Applications-Fundamentals) - Core statistical concepts and their applications in AI
- [0102-Beyond-LLMs-Prompts-Agents-RAG](./0102-Beyond-LLMs-Prompts-Agents-RAG) - Advanced topics in large language models, prompt engineering, agents, and retrieval-augmented generation
- [0103-Deep-Learning-Fundamentals](./0103-Deep-Learning-Fundamentals) - Foundational concepts in deep learning, including neural networks, activation functions, optimizers, and training techniques

### 02 - Frameworks & Responsible AI
- [0201-RAI-Governance-and-Ethical-Fundamentals](./0201-RAI-Governance-and-Ethical-Fundamentals) - Responsible AI governance, ethics, fairness, and inclusion principles
- [0202-PyTorch-Fundamentals](./0202-PyTorch-Fundamentals) - Core PyTorch concepts for building and training machine learning models
- [0203-TensorFlow-Fundamentals](./0203-TensorFlow-Fundamentals) - Core TensorFlow concepts for building and training machine learning models

Each directory contains detailed study materials with theoretical explanations, practical examples, and code snippets to support deep understanding of the respective topics.
